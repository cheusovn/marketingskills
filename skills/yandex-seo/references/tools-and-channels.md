# Инструменты Яндекса и каналы продвижения в Рунете

## Обязательные инструменты Яндекса

### Яндекс Вебмастер
Центр управления индексацией и диагностикой.
- **Индексирование:** статус страниц, исключённые URL и причины, статистика обхода.
- **Переобход страниц:** ускоренная отправка приоритетных URL в очередь индексации (лимиты по дням).
- **Региональность:** назначение региона сайта/разделов (см. regional-promotion.md).
- **Турбо-страницы:** настройка через RSS/YML/API, статистика.
- **Диагностика и безопасность:** фатальные/критичные ошибки, заражения, фильтры.
- **Sitemap и robots.txt:** добавление и проверка.
- **Запросы:** показы, клики, CTR, позиции по запросам и страницам.
- **Внешние ссылки:** профиль ссылающихся сайтов.

### Яндекс Метрика
Аналитика поведения (критична для ПФ).
- **Вебвизор** — запись сессий пользователей.
- **Карты** кликов, скроллинга, ссылок, аналитика форм.
- **Цели и конверсии**, электронная коммерция.
- **Отчёты по источникам**, регионам, устройствам, поисковым запросам.
- Используй для поиска страниц с высоким отказом / низким временем — кандидатов на улучшение ПФ.

### Яндекс Wordstat
Сбор и оценка спроса.
- Частотность запросов, похожие и сопутствующие запросы.
- Разбивка по регионам и по сезонности/истории.
- Учитывай типы соответствия (операторы `"..."`, `!`, `+`) для точной частотности.

### Яндекс Бизнес
Локальное SEO и Карты.
- Карточка организации, отзывы, фото, прайс, услуги.
- Приоритетное размещение (платное) в локальной выдаче и Картах.
- Статистика: показы, маршруты, звонки, переходы.

### Яндекс Директ
Контекстная и медийная реклама.
- Быстрый трафик и проверка конверсионности посадочных, пока SEO растёт.
- Геотаргетинг для локального бизнеса.
- Данные по конверсионным запросам полезны для SEO-семантики.

## Дополнительные каналы Рунета

SEO в Яндексе усиливается присутствием на популярных в России площадках — это даёт трафик, упоминания бренда и поведенческие сигналы.

- **Дзен (Yandex/VK)** — контент-платформа с большим охватом; статьи и видео для информационного спроса.
- **VK (ВКонтакте)** — крупнейшая соцсеть РФ; сообщество, таргет, VK Реклама, мини-приложения.
- **Telegram** — каналы и боты; высокововлечённая аудитория, Telegram Ads.
- **RuTube / VK Видео** — видеохостинг для российской аудитории.
- **Маркетплейсы** (Wildberries, Ozon, Яндекс Маркет) — отдельный канал продаж и спроса для товарного бизнеса; у них своя внутренняя оптимизация (SEO карточек).
- **Справочники и агрегаторы:** 2ГИС, Zoon, профильные отраслевые каталоги — для локального бизнеса и NAP-сигналов.

## Сторонние SEO-инструменты для Рунета
- **Топвизор, SE Ranking, Rush Analytics, Serpstat, Key Collector** — съём позиций в Яндексе, сбор/кластеризация семантики, аудит.
- **Сравнение с конкурентами:** анализ топ-10 Яндекса по приоритетным запросам в нужном регионе.

## Реалии рынка (учитывать в стратегии)
- Доступность ряда западных сервисов и платёжных систем в РФ ограничена — при настройке аналитики/оплаты ориентируйся на российские решения (Метрика, СБП, карты «Мир»).
- Мобильный трафик доминирует — приоритет mobile-first во всём.
